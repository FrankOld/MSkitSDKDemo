## 简述

通过集成 SDK 可以在 App 内快速集成漫短剧看剧功能，快速获得 App 流量的商业化变现。

## 引入

```
allprojects {
    repositories {
        mavenCentral()
    }
}
```

在子项目build.gradle的dependencies中根据需求引入依赖:
```
implementation 'io.github.frankold:webcore:1.0.2' //最新版本号

```
